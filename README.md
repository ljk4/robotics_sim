## 3. 正向运动学 FK

### 3.1 问题描述

给定关节角度向量 **q** ∈ R^9，求解末端执行器的世界坐标 **p_ee** ∈ R^3。

### 3.2 方法一：MuJoCo 内部计算

调用 `mj_forward()` 计算全部连杆的世界位姿，直接读取末端位置：

```python
data.qpos[...] = q
mujoco.mj_forward(model, data)
# p_ee = 连杆原点 + 旋转矩阵 @ 局部偏移
p_ee = data.xpos[body_id] + data.xmat[body_id] @ offset_local
```

### 3.3 方法二：解析变换矩阵连乘

从骨盆到末端，逐连杆计算 4x4 齐次变换矩阵之积：

```
T_world_ee = T_world_pelvis * T_torso * T_shoulder_pitch * ... * T_elbow

其中 T_parent_child = | R_quat * R_joint(q)   pos_body |
                       | 0      0      0         1     |
```

**MuJoCo 运动学关键公式（经实验验证）：**

```
xpos[child]  = xpos[parent] + xmat[parent] * pos_body
xmat[child]  = xmat[parent] * R_quat(quat_child) * R_joint(q)
                                                        ^^^^^^^^
                                    注意: 先 body quat, 后 joint rotation!
```

该旋转顺序是经过严格测试验证的关键发现——错误的 `R_joint * R_quat` 顺序会导致旋转误差达 0.7。

### 3.4 FK 验证

50 组随机关节角度对比：

| 方法 | 位置误差 |
|------|---------|
| MuJoCo FK vs 解析法 | 最大 **8.18e-15 m** |
| 结论 | 完全一致（数值精度极限） |

## 4. 逆向运动学 IK

### 4.1 问题描述

给定末端目标世界坐标 **p_target**，求解最优关节角度：

```
min  || FK(q) - p_target ||^2
s.t. q_lower <= q <= q_upper
```

### 4.2 方法一：雅可比阻尼最小二乘法 DLS

**核心迭代公式：**

```
q_{k+1} = q_k + J^T (J J^T + lambda^2 I)^{-1} e
```

其中：
- **J** ∈ R^{3xn}：末端执行器的点雅可比矩阵
- **e** = p_target - p_current：位置误差向量
- **lambda**：阻尼因子（防止奇异位形下发散）

**点雅可比计算（含末端偏移修正）：**

```
J_point = J_trans - skew(r_world) * J_rot

r_world = xmat[body] @ offset_local           # 偏移量的世界坐标
skew(r) = |  0   -rz   ry |
          |  rz   0   -rx |                    # 反对称矩阵
          | -ry   rx   0  |
```

末端不在连杆原点（有 0.28m 偏移），忽略旋转分量会导致 IK 误差 > 50cm。

**算法步骤：**
1. 初始化 q = 0（Home 姿态）
2. `mj_forward()` + `mj_jacBody()` 计算雅可比
3. 构建点雅可比：J = jacp - skew(r_world) * jacr
4. DLS 更新：Δq = J^T (J J^T + λ^2 I)^{-1} e
5. q ← clip(q + Δq, q_lower, q_upper)
6. 收敛条件：||e|| < tol，否则返回步骤 2

**特点：** 收敛快（~2ms），但局部方法可能陷入"背后解"

### 4.3 方法二：数值优化法 L-BFGS-B

**求解策略：**

1. **优化算法**：L-BFGS-B（有限内存 BFGS，支持边界约束）
2. **全局搜索**：12 个随机起点并行求解，选取最优
3. **代价函数**：`cost(q) = ||FK(q) - p_target||^2`

**特点：** 鲁棒性强（成功率 100%），但较慢（~100ms）

### 4.4 IK 验证结果

| 测试项 | 雅可比 DLS | 数值优化法 |
|--------|-----------|-----------|
| 100 组随机目标 | 成功率 99% | 成功率 **100%** |
| 平均跟踪误差 | 0.07 mm | 0.07 mm |
| 单次求解耗时 | ~2 ms | ~100 ms |
| 前伸目标 | 偶尔局部最优 | 始终正确 |

**结论：** 雅可比法适合实时控制（快），优化法适合离线规划（鲁棒）。

## 5. 动力学

### 5.1 运动方程

```
M(q) * qddot + C(q, qdot) + G(q) = tau

M(q)   : 质量矩阵 (9x9, 对称正定)
C(q,qdot): 科氏力 + 离心力
G(q)   : 重力力矩
tau    : 关节驱动力矩
qddot  : 关节加速度
```

### 5.2 质量矩阵 M(q)

从 MuJoCo `mj_fullM()` 提取，在随机关节角度下分析：

| 属性 | 数值 |
|------|------|
| 条件数（零姿态） | 6.1（非常良态） |
| 条件数范围 | [6.6, 15.7]（全域良态） |
| 对称性误差 | 0.00（精确对称） |
| 最小特征值 | 0.11 > 0（正定） |
| 非对角耦合占比 | < 10%（上肢耦合弱） |

### 5.3 逆动力学 ID

**问题：** 给定运动 (q, qdot, qddot)，求所需力矩 tau。

```python
# 方法一: MuJoCo mj_inverse
data.qacc[:] = qddot
mujoco.mj_inverse(model, data)
tau = data.qfrc_inverse[...]

# 方法二: 手动公式
bias = mj_inverse(qddot=0)     # = C(q,qdot) + G(q)
tau = M @ qddot + bias
```

**验证：** 双方法误差 < 2.22e-16 N·m（数值精度）

### 5.4 正动力学 FD

**问题：** 给定力矩 tau 和状态 (q, qdot)，求加速度 qddot。

```python
# 方法一: MuJoCo mj_forward
data.qfrc_applied[:] = tau
mujoco.mj_forward(model, data)
qddot = data.qacc[...]

# 方法二: 手动公式
qddot = M^{-1} @ (tau - bias)     # 使用 scipy.linalg.solve
```

**验证：** 双方法误差 < 5.68e-14 rad/s^2（数值精度）

### 5.5 重力分析

零姿态下各关节重力力矩：

| 关节 | 重力力矩 (N·m) | 原因 |
|------|---------------|------|
| torso | 0.000 | Z 轴旋转不受重力影响 |
| shoulder_pitch | -1.097 | 手臂自然下垂产生 |
| elbow | -1.044 | 前臂质量产生 |

躯干绕 Z 轴旋转**不改变重力方向** → 重力力矩恒定。

### 5.6 科氏力分析

同一姿态，不同速度下的科氏力贡献：

| 速度 (rad/s) | 重力 norm | 科氏力 norm | 科氏力占比 |
|-------------|----------|------------|----------|
| 0.5 | 2.14 | 0.23 | 11% |
| 1.0 | 2.14 | 0.91 | 48% |
| 2.0 | 2.14 | 3.65 | **111%** |

**结论：** 高速运动时科氏力可超过重力，必须纳入控制模型。

### 5.7 PD + 重力前馈控制器

```
tau = Kp * (q_des - q) + Kd * (qd_des - qd) + G(q)
       |____比例____|     |____阻尼____|     |_前馈_|
```

- **比例项** Kp·Δq：弹簧效应，驱动关节朝向目标
- **微分项** Kd·Δqd：阻尼效应，抑制震荡（Kd ≈ Kp/10）
- **重力前馈** G(q)：精确抵消重力，消除稳态误差

**有无重力补偿比较：**

| 配置 | 稳态误差 |
|------|---------|
| 带重力补偿 | ~0 mrad |
| 无重力补偿 | e_ss = G(q)/Kp ≈ 2~7 mrad |

### 5.8 ID → FD 闭环验证

```
原始加速度 qddot_original
  → ID: tau = ID(q, qdot, qddot_original)
  → FD: qddot_recovered = FD(q, qdot, tau)
  → 比较: ||qddot_recovered - qddot_original||

结果: 最大误差 < 9e-16 rad/s^2 —— 闭环完全一致
```

## 6. 仿真演示

`simulation_demo.py` 提供交互式 MuJoCo 3D 仿真演示，涵盖运动学和动力学：

### 6.1 演示功能

**运动学：**
- **IK**：L-BFGS-B 数值优化从末端目标求解关节角度
- **FK**：MuJoCo 正向运动学实时计算末端位置，终端显示跟踪误差

**动力学：**
- **力矩分解**：实时显示惯性力矩、科氏力、重力力矩三分量的范数和各关节值
- **ID 验证**：按 `I` 键运行 ID 双方法验证（mj_inverse vs M·qdd + bias）
- **FD 验证**：按 `I` 键运行 FD 双方法验证（mj_forward vs M^{-1}(tau - bias)）
- **质量矩阵**：按 `I` 键显示当前 M(q) 的条件数和行列式
- **重力补偿对比**：按 `G` 键切换重力前馈 ON/OFF，观察稳态误差变化

### 6.2 演示流程

```
Home → 右臂上举 → 左臂前伸 → 双臂前伸 → T型平举 → 不对称姿态 → 循环
```

每个姿态的步骤：
1. **IK 预求解** → 计算目标关节角度
2. **轨迹生成** → 五阶多项式平滑过渡
3. **PD+重力补偿** → 关节空间轨迹跟踪
4. **FK 实时验证** → 终端显示末端误差

### 6.2 轨迹生成：五阶多项式

```
s(t) = 10t^3 - 15t^4 + 6t^5       (t in [0,1])

边界条件:
  s(0) = 0,   s(1) = 1      (位置连续)
  s'(0) = 0,  s'(1) = 0     (速度连续, 零启停)
  s''(0) = 0, s''(1) = 0    (加速度连续, 无力矩冲击)
```

### 6.4 键盘交互

| 按键 | 功能 |
|------|------|
| `N` | 跳到下一个目标姿态 |
| `G` | 切换重力补偿 ON/OFF（观察稳态误差变化） |
| `D` | 切换动力学详情显示（力矩三分量分解） |
| `I` | 运行 ID/FD 双方法验证 + 显示质量矩阵属性 |
| `SPACE` | 暂停/继续 |
| `ESC` | 退出 |

### 8.3 关键技术发现

1. **MuJoCo 旋转顺序**：`xmat[child] = xmat[parent] * R_body_quat * R_joint`（先 body quat 后 joint rotation），经实验严格验证。

2. **点雅可比修正**：末端执行器有 0.28m 偏移，忽略旋转分量导致 IK 误差 > 50cm。

3. **雅可比 DLS vs L-BFGS-B**：前者快速（2ms）但局部；后者鲁棒（100ms）但较慢。互补使用。

4. **重力补偿必要性**：纯 PD 控制有固有稳态误差 e_ss = G(q)/Kp，重力前馈可消除。

5. **科氏力**：高速运动（qdot=2 rad/s）时科氏力可达重力的 1.1 倍，不可忽略。
